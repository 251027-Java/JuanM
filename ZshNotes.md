# Zsh Notes

## Basic Zsh Commands
- pwd - Print Working Directory
- cd - Change Directory
- mkdir - Make Directory
- touch - Create a File
- ls - List Contents of a Directory
	- -a - List All (including hidden files)
	- -l - List in a verbose format
- rm - Remove File/Directory
	- -r - Remove recursively
	- -f - Force
- nano - Open the Nano Text Editor
- rmdir - Remove Directory
- vim - Open the Vim Text Editor

## File Paths
~ - Home Directory
- Absolute Path - file path starting with / that specifies a unique location
- Relative Path - file path starting with . that defines the relation to the current working directory
	- .. - used with Relative Paths to denote the parent directory
